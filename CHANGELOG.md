# Changelog

## 0.1.0

- Initial experimental Flutter package foundation.
- Added BDK adapter boundary for future `bdk-dart` integration.
- Added secure storage, memory storage, sync state, balance, receive address, transaction, and onboarding models.
- Added basic Flutter widgets for balance, receive address, and sync state.
- Added an example Flutter app demonstrating the intended developer experience.
- Added tests for the initial public API.
